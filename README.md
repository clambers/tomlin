Tomlin
======

Tomlin is a C++ parser for TOML. It's currently working on full compliance with
the v0.1.0 specification.

Current v0.1.0 Compliance
-------------------------

| Feature  | Supported? |
| :------- | :--------: |
| Comment  | no         |
| String   | no         |
| Integer  | yes        |
| Float    | yes        |
| Boolean  | no         |
| Datetime | no         |
| Array    | no         |
| Hash     | no         |
